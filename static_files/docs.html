Obtener resultados del Test CHAEA en .json

1. Crear el Test

Añada una actividad a Moodle que siga el siguiente esquema:

Este cuestionario deberá ser llenado por los alumnos, quienes deben completarlo con los resultados que obtuvieron al realizar el test CHAEA.

2. Exportar los resultados

Una vez que los alumnos hayan completado el formulario, ingrese a la pestaña de respuestas de la actividad.
La manera de acceder a esta pestaña puede variar según la versión de Moodle que esté utilizando.

A continuación, verá una opción llamada "Descargar datos de tabla como". Despliegue la lista y selecciones "Javascript Object Notation".
Finalmente, descargue el archivo presionando "Descargar".
